# Project Goals: Bio-R&D Proposal Dashboard UI/UX

본 문서는 바이오 국가과제 수주 연구기획 대시보드 UI/UX 개선 프로젝트의 최종 목표와 MVP(최소 기능 제품) 범위를 정의합니다.

## 1. 최종 목표 (Final Goal)
- **목표**: 기존 바이오 R&D 기획 대시보드(`index.html`)의 시각 디자인을 프리미엄 수준(글래스모피즘, 커스텀 그라데이션, 세련된 타이포그래피)으로 개선하고, 사용자 중심의 직관적인 레이아웃과 동적인 사용자 흐름(Flow)을 정립하여 완벽하게 동작하는 프론트엔드 환경을 구축합니다.
- **주요 가치**: 비주얼 임팩트 극대화(Premium Aesthetics), 실무 툴박스 기능 실질화(Interactive Tools), 크로스 브라우저 호환성 및 화면 안정성(Verification).

---

## 2. MVP 개발 범위 (MVP Scope)

### 2.1 UI/UX 및 디자인 개선 (포함 사항)
- **디자인 시스템 구축**: 프리미엄 테마(Dark/Light 하이브리드), 통일감 있는 색상 스키마(산학연병 주체별 고대비 컬러 매핑), 현대적 서체(Outfit/Inter) 조화.
- **글래스모피즘 레이아웃**: 반투명 카드, 부드러운 드롭 섀도우, 마우스 반응형 호버 애니메이션.
- **컴포넌트 폴리싱**: 버튼 디자인 개선, 아이콘 가시성 보완, 마크다운 뷰어 영역의 여백 및 가독성 개선.

### 2.2 사용자 흐름 및 버튼 배치 최적화 (포함 사항)
- 사이드바 메뉴 활성화 효과 개선 및 문서 로드 흐름 개선.
- 툴박스(자가진단, 예산 시뮬레이터, TRL 로드맵)의 입력 폼(Input)과 조작 버튼(Action Button)을 화면 하단이나 적절한 카드 내 우측으로 배치하여 시선의 흐름과 동선 일치.

### 2.3 동적 기능 실질화 (포함 사항)
- **예산 시뮬레이터**: 예산 한도 초과(예: 위탁비 40% 초과)나 학생 인건비 비율에 따른 실시간 시각 경고 게이지 바 구현.
- **자가평가 진단**: 진단 항목 체크 시 실시간 합격/점수 진단 게이지 및 결과 요약 카드 노출.
- **TRL 로드맵**: 마일스톤 단계별(1~5단계) 상세 정보 호버 툴팁 및 단계 전환 효과 구현.

---

## 3. 제외 기능 (Excluded Features)
- **데이터베이스 연동**: 별도의 서버 사이드 SQLite 데이터베이스 연결 및 저장 기능은 MVP 범위에서 제외하며, 순수 프론트엔드 로컬 상태값(State)으로 처리합니다.
- **실시간 PDF/RFP 파서 백엔드**: PDF를 직접 업로드하여 OCR로 텍스트를 추출하는 백엔드 AI 기능은 제외하며, 기 정의된 RFP 예시 텍스트/마크다운 파싱으로 대체합니다.

---

## 4. 향후 확장 범위 (Future Expansion)
- **로컬 LLM (Ollama) 연동**: 후보물질 보안을 위한 로컬 AI 기반 개조식 연구 내용 자동 작성 기능.
- **DOCX 내보내기 백엔드**: 작성 완료된 마크다운을 HWP 호환 스타일 DOCX 파일로 직접 컨버팅하는 파이썬 모듈 연동.

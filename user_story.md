# User Stories: Bio-R&D Proposal Dashboard

본 문서는 바이오 국가과제 수주 연구기획 대시보드를 사용하는 사용자의 경험 시나리오를 정의합니다.

## 1. 페르소나 (Persona)
- **김총괄 (52세, 산학연병 컨소시엄 총괄 PM / 대학 교수)**
  - 역할: 바이오 국가과제 주관책임자로서 전체 연구계획 및 추진체계를 관리합니다.
  - 고충: 다양한 공동기관(연구소, 기업, 병원)의 서류와 일정을 조율하기 복잡하고, 국가과제 양식 및 규정을 맞추는 데 시간이 많이 소요됩니다.
  
- **이지혜 (31세, 바이오 벤처기업 R&D 기획 연구원)**
  - 역할: 특허성(FTO) 분석, 예산 배분 실무 및 연구계획서 직접 작성을 담당합니다.
  - 고충: 혁신법에 부합하는 예산 편성 규정이 매번 헷갈리고, 평가위원 사전검증(IRB/IACUC 등) 체크리스트를 누락 없이 챙기기가 어렵습니다.

---

## 2. 사용자 스토리 (User Stories)

### 2.1 사이드바 네비게이션 및 레이아웃
- **스토리**: R&D 기획자로서, 나는 대시보드 내의 다양한 기획 가이드와 툴박스를 직관적으로 이동하고 싶다. 그래야 필요한 챕터나 기능을 헤매지 않고 빠르게 찾을 수 있기 때문이다.
- **수용 기준 (Acceptance Criteria)**:
  - 사이드바 메뉴 클릭 시 해당 화면 또는 가이드가 우측 메인 공간에 부드럽게 로드된다.
  - 현재 활성화된 메뉴 아이템이 시각적으로 명확하게 강조(Highlight)된다.

### 2.2 실무 지원 툴박스 조작 및 시각화
- **스토리**: R&D 실무 연구원으로서, 나는 연구비 예산을 모의 배분하거나 TRL 마일스톤 단계를 시뮬레이션해보고 싶다. 그래야 계획서 작성 전에 과제 적정성을 검증하고 예산 에러를 미연에 방지할 수 있기 때문이다.
- **수용 기준 (Acceptance Criteria)**:
  - 예산 시뮬레이터에서 모달리티(항체, 세포치료제 등) 선택과 총 연구비 입력 시, 혁신법에 맞춘 권장 예산 비율이 실시간 차트/게이지로 업데이트된다.
  - TRL 로드맵 뷰어에서 연차별 마일스톤 단계가 단계별(TRL 1~5)로 가시성 높게 표시된다.
  - 모의 자가진단 항목 체크 시, 전체 진단 진척도 또는 합격 기준 점수가 반영된다.

### 2.3 계획서 근거 매핑 및 뷰어 가독성
- **스토리**: 총괄 PM으로서, 나는 계획서 작성 지침과 챕터별 세부 요건을 큰 화면에서 편안하게 읽고 싶다. 그래야 각 챕터의 핵심 기획 요건을 명확히 파악할 수 있기 때문이다.
- **수용 기준 (Acceptance Criteria)**:
  - 마크다운으로 작성된 장별 가이드북(`docs/plans/designs/handbook/`)이 깔끔한 타이포그래피와 충분한 대조비(Contrast)를 갖춘 프리미엄 뷰어로 렌더링된다.
  - 뷰어 내의 관련 파일/코드 링크 클릭 시 해당 문서로 매끄럽게 연결된다.
